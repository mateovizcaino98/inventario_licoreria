## Descripción del sistema

Este proyecto consiste en el diseño de un sistema de gestión de inventario para una licorería. 
El programa consta de 3 sprints. En esta primer sprint se desarrolla la lógica del programa mediante un diagrama de flujo y su respectivo pseudocódigo. 
El sistema permitirá administrar los productos del inventario de forma organizada y servirá como base para las siguientes etapas del proyecto.

## Funciones principales

- Mostrar la lista de productos.
  - Ver todos los productos.
  - Buscar un producto específico.

- Añadir un producto.

- Actualizar un producto.
  - Registrar una venta (reducir la cantidad disponible).
  - Actualizar todos los datos de un producto.

- Salir del sistema.
